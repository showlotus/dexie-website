Table.schema

  Edit on GitHub

Syntax

db.[tableName].schema

Type

TableSchema

Sample

var db = new Dexie("MyDB");
db.version(1).stores({friends: "++id,name"});

alert ("Primary key: " + db.friends.schema.primKey.keyPath); // Will alert ("id");
